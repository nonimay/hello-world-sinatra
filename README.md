hello-world-sinatra
-------------------
Hello World Sinatra app using Tilt as the template engine.

To run the application:
```
rackup
```
OR
```
rackup -p [port]
```

To tun the tests:
```
rake test
```

Uses CircleCI for continuous integration and deployed to Heroku at
<https://desolate-tor-97765.herokuapp.com/>
